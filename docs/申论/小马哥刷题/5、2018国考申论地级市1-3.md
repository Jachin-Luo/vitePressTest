#### 六、应用文写作核心原则 ﻿01:36:15﻿

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/166ee02cf7a44a4f145b17715237597f.jpeg)

- 实质功能优先: 应用文的核心是"用来用的文"，写作时必须明确目的和实际用途，而非纠结形式细节。如圣保罗案例所示，城市设计需恢复公共领域交往功能才能解决根本问题。

- 三要素框架

  :

  - 格式规范: 根据文种类型确定标题、称谓、落款等要素
  - 结构完整: 不同文种有对应的行文结构要求
  - 内容充实: 需包含实质性信息而非空话套话

#### 七、应用文格式逻辑解析

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/6d0ec01d45a42cae706d8e7cbb7f656e.jpeg)

- 称谓必要性

  :

  - 需要情形: 通知、倡议书等需明确告知对象的文种，如"各位同学"
  - 无需情形: 新闻稿、短评等面向公众的文种

- 落款逻辑

  :

  - 需要情形: 需表明责任主体的正式文书（如通告）或发起者（如倡议书）
  - 无需情形: 现场发言稿（发言者已在场）或媒体稿件（编辑负责制）

- 标题通用性

  :

  - 例外情况: 编者按、内容要点等附属文本可不设标题
  - 记忆技巧: 通过现实场景推演（如"人民日报新闻稿是否有落款"）替代死记硬背

#### 九、应用文 ﻿01:42:20﻿

##### 1. 基本要素

- 称谓选择：根据受众群体确定，如"全县广大人民群众"适用于全县范围的倡议书，讲话稿可省略称谓
- 落款原则：实用主义原则，有用则写（如倡议书需注明发起单位），无用则免（如现场讲话已知发言人身份可不写）
- 时间标注：重要时效性文书必须注明日期（如倡议书），非时效性文书可省略

##### 2. 结构框架

- 标题规范：采用"关于+事由+文种"格式，例："关于文明发言的宣传稿"

- 正文组成

  ：

  - 背景/意义（简明扼要）
  - 核心对策（占主要篇幅）
  - 结尾号召（字数不足时可省略）

- 格式弹性：河南省等地区不强制要求完整格式，以内容实用性为主

##### 3. 写作要点

- 内容侧重：80%篇幅应聚焦具体做法（如花椒酱案例需突出实操经验）

- 表达要求

  ：

  - 背景分析要高度概括
  - 原因阐述需简化处理
  - 对策建议须详细具体

- 字数控制：严格遵循题目要求，超出部分扫描时无法识别

##### 4. 地域特色

- 格式差异：河南省明确不要求格式规范，近年更多省份采用此趋势
- 评分重点：区分度在于内容质量而非格式完整度
