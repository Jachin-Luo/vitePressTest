### 严谨逻辑关系的一般质疑

| a    | b    | a->b |
| ---- | ---- | ---- |
| 真   | 真   | 真   |
| 真   | 假   | 假   |
| 假   | 真   | 真   |
| 假   | 假   | 真   |

a->b 

质疑为:a—>-b







充分必要条件

箭头前面表示充分条件，后面表示必要条件，"充分条件→必要条件"

充分条件

关键词





必要条件

关键词

基础、前提、关键、必须、需要、离不开、必不可少、不可或缺、"必然要求"等词语都表示必要条件

示例

"想考上必须努力" → "考上→努力"
