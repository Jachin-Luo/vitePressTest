#### 二、知识小结

| 知识点       | 核心内容                                                                 | 考试重点/易混淆点                            | 难度系数 |
| ------------ | ------------------------------------------------------------------------ | -------------------------------------------- | -------- |
| 两期比重概念 | 现期比重与基期比重的差值，反映占比变化趋势                               | 区分“比重差”与“增长率”                       | ⭐⭐     |
| 题型识别     | 出现两个时间（如 2024 年 vs 2023 年）、占比比较、选项为百分点            | 注意“占”字和“比重”关键词                     | ⭐⭐     |
| 比较技巧     | a（部分增速）>b（整体增速）→ 比重上升；反之下降；相等则不变              | 带正负号比较增长率（如-0.2% > -1.5%）        | ⭐⭐⭐   |
| 计算公式     | 现期比重-基期比重 = (A/B) × (a-b)/(1+a)                                  | 优先计算 a-b 判断升降，再估算数值大小        | ⭐⭐⭐⭐ |
| 速算结论     | 若 a>0，两期比重差绝对值 < \|a-b\|                                       | 选项排除法：先排除不符升降的选项，再选更小值 | ⭐⭐⭐   |
| 逆运用       | 比重上升 →a>b；比重下降 →a<b                                             | 通过占比变化反推增速关系（如例 1-3）         | ⭐⭐⭐   |
| 无增速处理   | 1. 直接算现期/基期比重差；2. 反推增长率再代入公式                        | 修正思维估算（如例 4-5）                     | ⭐⭐⭐⭐ |
| 易错陷阱     | 1. 忽略增长率正负；2. 混淆部分/整体数据；3. 数量级错误（如 10% vs 0.1%） | 重点检查数据对应关系                         | ⭐⭐⭐⭐ |
