##### 1. 现期倍数

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/de33a057b447e61dffaa90fd78ad9201.jpeg)

- 题型识别：当问题时间与材料一致，出现"A是B的几倍"、"A比B增长/多/高几倍"时使用现期倍数计算

- 基础公式

  ：倍数

  ﻿=AB= \frac { A } { B }=BA﻿

- 特殊情形处理

  ：

  - 增长倍数

    ：A比B增长/多/高几倍时，公式为

    ﻿A−BB=AB−1\frac { A - B } { B } = \frac { A } { B } - 1BA−B=BA−1﻿

  - 超过倍数

    ：A超过B的n倍表示为

    ﻿AB>n\frac { A } { B } > nBA>n﻿
