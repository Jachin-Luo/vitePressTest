| 速算技巧：百化分 | 将增长率转化为分数形式（如 r=1/n）：- 正增长率：增长量=现期量 ÷(n+1)- 负增长率：减少量=现期量 ÷(n-1)口诀：正加一，负减一                                                                                    | 百分数与分数转换（如 6.7%≈1/15）选项差距大时估算优先                     | ⭐⭐⭐   |
| 速算技巧：直除   | 当百化分不适用时，直接计算：现期量 ÷(1+增长率)× 增长率（保留三位数快速估算）                                                                                                                                | 首位商法（快速判断选项范围）单位换算（如亿 → 万亿需左移四位）            | ⭐⭐     |
| 多位数和差模型   | 多个现期和与基期和做差时，分项对应做差再求和（如(A₁-A₂)+(B₁-B₂)），减少计算量                                                                                                                               | 避免整体求和再减（易出错且耗时）                                         | ⭐⭐     |
| 同环比陷阱       | 注意题目要求是同比（与去年同期比）还是环比（与上月比），如第 16 题需用环比增长率                                                                                                                            | 关键词识别（“较上月”vs“较上年”）                                         | ⭐⭐⭐   |
| 高增长率处理     | 当增长率>100%时：- 百化分转化（如 110%≈1/0.91）- 直除简化（现期量 ÷(1+增长率)× 增长率 ≈ 现期量 ÷2）                                                                                                         | 近似估算（如 108%≈1.1 倍）选项验证（优先排除明显偏差项）                 | ⭐⭐⭐⭐ |
| 部分量计算       | 现期量未直接给出时，需通过占比反推（如第 19 题：社会消费品总额 ×25%=实物商品零售额）                                                                                                                        | 比重与总量联动单位一致性检查                                             | ⭐⭐⭐   |
