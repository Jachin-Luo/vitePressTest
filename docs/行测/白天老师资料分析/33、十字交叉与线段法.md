##### 4. 十字交叉与线段法的解法 ﻿05:25﻿

-   ![img](https://jachin-img.s3.bitiful.net/2025/12-10/d9bc3ff6c996694d7db5cd45af05c17a.jpeg)

-   代数推导

    ：

    -   从

        Aa+Bb=(A+B)cAa + Bb = (A+B)cAa+Bb=(A+B)c

        出发

    -   展开得

        Aa+Bb=Ac+BcAa + Bb = Ac + BcAa+Bb=Ac+Bc

    -   移项得

        A(a−c)=B(c−b)A(a-c) = B(c-b)A(a−c)=B(c−b)

    -   最终得比例关系

        AB=c−ba−c\frac{A}{B} = \frac{c-b}{a-c}BA=a−cc−b

-   十字交叉法操作

    ：

    -   将 a、b 写在两侧，c 写在中间
    -   交叉做差（大数减小数）得 a-c 和 c-b
    -   量比等于差值的反比

-   线段法操作

    ：

    -   将 a、b 标在线段两端，c 标在中间
    -   计算 a-c 和 c-b 的距离
    -   应用"量与距离成反比"原则

    ）

-   速算技巧

    ：

    -   居中原则：混合后整体增长率介于部分增长率之间但不正中

    -   偏大原则：整体增长率偏向基数较大部分的增长率（基数本质是基期量，但通常用现期量近似代替）

    -   线段法

        ：

        AB=b−cc−a\frac{A}{B} = \frac{b-c}{c-a}BA=c−ab−c

        ，其中量与距离成反比

-   本质原理

    ：

    -   基期量关系：混合前后增长量不变，通过基期量 × 增长率建立等式

    -   模型推导

        ：

        大 a× 小 a+大 b× 小 b=(大 a+大 b)× 小 c 大 a× 小 a + 大 b× 小 b = (大 a+大 b)× 小 c 大 a× 小 a+大 b× 小 b=(大 a+大 b)× 小 c

        ，其中大 a、大 b 代表基期量

    -   近似处理：实际解题中可用现期量代替基期量简化计算
