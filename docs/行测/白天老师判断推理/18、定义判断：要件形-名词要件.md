##### 1. 定义判断考情介绍 ﻿00:40﻿

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/1be02fe0de4f9e5601018bcbb67bd920.jpeg)

- 题量分布: 国考中定义判断通常为10道题，占判断推理总题量（30-40道）的四分之一

- 命题形式: 题干给出词语定义解释，选项给出具体实例，要求判断最符合/不符合（属于/不属于）定义的选项

- 选项设置: 均为单选题，需择优选择（可能出现多个部分符合的选项）

- 题型分类

  :

  - 单定义：仅给出一个词语的定义
  - 多定义：给出多个词语的定义（本质是多个单定义的组合）

- 注意事项

  :

  - 必须看清提问要求"属于"还是"不属于"（错误率显著升高点）
  - 提问设坑是常见命题陷阱，需养成先看提问的做题习惯

##### 2. 单定义解题框架

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/7b43cb0cb3a551bc3b819f5ffd9850a9.jpeg)

- 两大类型

  :

  - 要件型: 通过提取题干关键词（要件信息）排除错误选项
  - 脉络型: 梳理题干信息逻辑关系选出正确答案

- 共同核心: 关键词的识别与运用（包括关系词也是关键词）

###### 1）名词型要件

- ![img](https://jachin-img.s3.bitiful.net/2025/12-10/e550e6d1aee9bdf799da451f78033654.jpeg)

- 三类核心名词

  :

  - 主体: 动作发出者（如"桑桑老师给同学们上课"中的"桑桑老师"）
  - 客体: 动作承受者（如"同学们"承受上课动作）
  - 属: 被定义项的所属大类（如"教育培训机构"是"上岸之路"的属概念）

- 应用规则

  :

  - 选项中的名词要件与题干不一致→直接排除
  - 主体/客体通常位于句首，应优先识别

- 解题优势

  :

  - 主客体识别速度快，可快速缩小选项范围
  - 属概念错误可直接判定选项不符合定义

- 例题:自发性秩序定义判断 

  30:03

  - 概念解析

    - 核心定义: 指在特定群体中存在的由众人行为自发形成并对个体成员具有内在影响或约束的社会习惯。

    - 属概念: 属于"社会习惯"这一大类，区别于法律规范等强制性规则。

    - 种差特征

      :

      - 形成方式：由众人行为自发形成
      - 作用机制：对个体具有内在影响或约束
      - 适用范围：特定群体内部

  - 关键要素分析

    - 特定群体范围: 可以是地域性群体（如某县、某村）、文化群体（如中国人）等，范围可大可小但需明确边界。
    - 自发形成过程: 非官方强制规定，而是通过群体成员间相互影响逐渐形成。
    - 约束表现形式: 既包括正面引导（如学习武术），也包括负面压力（如被看不起）。

  - 例题解析

    - 题目类型: 选非题（选择"不属于"的选项）

    - 选项分析

      :

      - A选项（武术之乡）：符合"众人自发形成"+"影响个体行为"的特征
      - B选项（"吃了吗"问候语）：体现文化习惯的自然习得过程
      - C选项（公交专用道）：属于法律强制规定，非自发形成的社会习惯
      - D选项（孝敬风气）：典型的社会习惯约束机制

    - 解题技巧

      :

      - 先确认属概念是否为"社会习惯"
      - 再验证是否满足"自发形成"和"内在约束"特征
      - 特别注意法律规范与习惯的区别

    - 正确答案: C选项

    - 易错警示: 容易混淆社会习惯与法律规范的约束来源，需注意题干问的是"不属于"的情形。

  - 方法论总结

    - 解题步骤

      :

      - 明确属概念范围（如本题的"社会习惯"）
      - 分析种差特征（形成方式、作用机制等）
      - 排除明显不符合属概念的选项
      - 对比剩余选项的种差符合度

    - 注意事项

      :

      - 特定群体的表述可能比较泛泛，不作为主要判断依据
      - 属概念题目往往需要结合种差特征综合判断
      - 法律规范与社会习惯的区分是高频考点
