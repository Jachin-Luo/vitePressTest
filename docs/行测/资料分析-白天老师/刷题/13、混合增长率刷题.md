-   居中原则：混合后整体增长率介于部分增长率之间（不一定是正中值）

-   偏大原则：整体增长率偏向基数较大部分的增长率（基数通常用现期量近似代替）

-   线段法

    ：量与距离成反比，公式为

    AB=b−cc−a\frac{A}{B} = \frac{b-c}{c-a}BA=c−ab−c

-   解题优先级：先判断居中性，再分析偏大性，最后考虑线段法计算
